Access Keys

The Voodoo Envy laptop comes with this power brick + wireless bridge adapter. Normally to connect to it you would download some silly 64 bit windows 7 driver, but really all the driver does is add a wireless connection profile.

The access point itself has the following connection settings:

  • Channel 6 (mine is)
  • WPA2 + AES Encryption
  • Essid: "Voodoo Power Adapter"
  • Password: "ABinkie!"
  • No quotes obviously

All information extracted from the "Wireless Network Connection-Voodoo Power Adapter.xml" file contained in the driver.

Teardown

This thing was the tightest piece of plastic I've ever disassembled. There are no screws, just really tight plastic. You will break it if you take it apart.
